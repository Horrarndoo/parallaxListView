# parallaxListView
	
	实现仿QQ控件主页效果：
	1.透明状态栏+TitleBar
	2.动态渐变色状态栏+TitleBar
	3.下拉拖拽视差效果
	4.下拉加载更多
	5.点击按钮弹出PopupWindow list选项效果
	6.点击+顶部弹出PopupWindow界面效果
博客链接：[Android自定义控件：打造自己的QQ空间主页](http://blog.csdn.net/oQinYou/article/details/68491295)
